## 3. 登录说明

1. 打开 `https://nxm.zlck888.com/`。
2. 在首页登录框输入登录名和密码。
3. 验证码是数学题，例如 `4 + 2 = ?`，填写 `6`。
4. 点击“登录系统”。
5. 登录后进入“运行概览”。

如果验证码一直加载：

1. 按 `Ctrl + F5` 强刷。
2. 仍不正常时，打开 `https://nxm.zlck888.com/api/auth/captcha`，应看到 JSON 验证码。
3. 接口正常但页面异常时，清理浏览器缓存后重试。

## 4. 超级管理员操作教程

### 4.1 首页管理

路径：

```text
后台 -> 首页管理
```

可维护内容：

- 品牌素材：Logo、站点图标 Favicon、Banner。
- 导航：系统名称、导航启用状态。
- 横幅：首页标题、说明、按钮文字和链接。
- 页脚：页脚文案。
- 列表内容：通知公告、资料下载、服务事项、导航链接、横幅状态、亮点数据。

### 4.2 上传 Logo / Banner / Favicon

路径：

```text
首页管理 -> 基础区域 -> 品牌素材
```

操作：

1. 点击“替换 Logo”“替换图标”或“替换 Banner”。
2. 选择文件上传。
3. 上传成功后刷新首页查看效果。

限制：

- Favicon 支持 `ico/png/svg/jpg/jpeg/webp`，大小不超过 512KB。
- Logo/Banner 支持常见图片格式。
- 未上传素材时系统会使用默认文字徽标或兜底样式。

### 4.3 编辑通知公告富文本

路径：

```text
首页管理 -> 列表内容 -> 通知公告 -> 编辑 -> 详情
```

富文本支持：

- 正文、H2、H3。
- 加粗、斜体、下划线。
- 无序列表、有序列表、引用。
- 插入链接。
- 上传并插入图片。
- 清除格式。

保存后，公开首页通知详情会显示清洗后的安全 HTML。系统会自动过滤脚本、危险链接和不安全图片。

### 4.4 安全中心

路径：

```text
后台 -> 安全中心
```

可查看：

- 登录限流策略。
- 白名单 IP。
- 临时放宽登录限流状态。
- 安全事件。
- 登录锁定/限流事件。

### 4.5 账号管理

路径：

```text
后台 -> 账号管理
```

常见操作：

- 新增用户。
- 启用/禁用用户。
- 重置密码。
- 按角色分配账号。

注意：普通管理员不要降级或删除超级管理员能力。

## 5. 单位用户操作教程

### 5.1 项目申报

路径：

```text
后台 -> 项目申报
```

基本流程：

1. 新增项目。
2. 填写项目标题、批次、单位、联系人、经费等信息；预算金额按“万元”录入，系统会同步显示保存金额（元）。
3. 上传附件材料。
4. 保存草稿。
5. 确认无误后提交审核。

### 5.2 全周期记录

路径：

```text
后台 -> 全周期管理
```

可处理：

- 合同任务书。
- 实施进展。
- 整改闭环。
- 专家认证相关记录。

项目选择器支持按项目名搜索，不需要手填项目 ID。

### 5.3 验收提交

路径：

```text
后台 -> 验收管理
```

操作：

1. 找到需要验收的项目。
2. 上传验收材料。
3. 按要求选择材料分类。
4. 提交验收。

如果批次配置了必传材料，缺失时系统会阻止提交并给出中文提示。

## 6. 区县/部门/专家操作教程

### 6.1 审核任务

路径：

```text
后台 -> 审核任务
```

可处理：

- 待审核项目。
- 已审核历史。
- 按当前角色查看对应阶段任务。
- 操作列点击“审核处理”进入审核弹窗；专家阶段会显示多维评分表。

### 6.2 验收管理

路径：

```text
后台 -> 验收管理
```

常用 tab：

- 待处理：当前需要处理的验收。
- 已处理：自己参与过或处理过的历史验收。
- 全部可见：在权限范围内可见的数据。

### 6.3 全周期查看

路径：

```text
后台 -> 全周期管理
```

可按项目查看合同任务书、实施进展、整改闭环和专家认证。
