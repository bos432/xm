# 业务操作教程

## 1. 首页访问

地址：

```text
https://nxm.zlck888.com
```

首页包含：

- 通知公告
- 服务事项
- 资料下载
- 申报批次
- 登录入口
- 新单位注册
- 忘记密码

当前首页已去除重复的旧“单位注册”入口，只保留“新单位注册”。

## 2. 登录

1. 打开首页。
2. 在登录框输入用户名和密码。
3. 根据验证码题目填写答案。
4. 点击登录。

示例：

```text
题目：6 + 7 = ?
答案：13
```

登录失败次数过多时，系统会显示中文风控提示和倒计时。

## 3. 单位用户操作流程

### 3.1 新单位注册

1. 首页点击“新单位注册”。
2. 填写单位名称、统一社会信用代码、联系人、手机号、邮箱、地址等信息。
3. 提交注册。
4. 等待管理员审核。
5. 审核通过后可登录申报。

### 3.2 创建项目

1. 单位账号登录。
2. 进入“项目管理”。
3. 点击“新增项目”。
4. 选择申报批次、项目类别、项目类型。
5. 填写项目名称、项目内容、预算、负责人等信息。
6. 保存草稿或提交。

### 3.3 上传附件

1. 在项目列表进入项目详情。
2. 打开附件或材料上传区域。
3. 按要求上传申报材料。
4. 确认材料完整后提交项目。

### 3.4 查看全周期

1. 在项目列表点击“全周期”。
2. 系统跳转到 `/lifecycle?project_id=<项目ID>`。
3. 页面自动回显项目名称、单位、批次、状态。
4. 可查看合同任务书、实施进展、整改闭环、专家认证等记录。

### 3.5 提交验收

1. 项目状态进入可验收阶段后，进入“验收管理”。
2. 点击“发起验收”。
3. 选择项目。
4. 上传验收材料。
5. 按材料分类选择：
   - 验收申请书
   - 项目总结
   - 财务材料
   - 成果证明
   - 其他
6. 提交验收。

### 3.6 处理整改

1. 如果项目或验收被要求整改，单位进入“全周期”或相关待办。
2. 查看整改要求和意见。
3. 上传整改材料并提交。
4. 等待管理员复核。

## 4. 区县审核角色操作

1. 使用区县账号登录。
2. Dashboard 查看“我的待审”“已审”“验收待审”“验收已审”。
3. 点击指标进入对应列表。
4. 打开项目或验收详情。
5. 查看申报材料、项目基础信息、全周期记录。
6. 填写审核意见。
7. 选择通过或退回。
8. 已处理记录可在“已处理”或“已审”中查看。

## 5. 部门审核角色操作

部门角色与区县类似：

1. 登录后进入 Dashboard。
2. 查看部门待审任务。
3. 进入项目详情。
4. 查看区县意见和申报材料。
5. 填写部门审核意见。
6. 提交通过或退回。
7. 在验收管理中切换“待处理 / 已处理”查看验收任务。

## 6. 专家评审角色操作

1. 使用专家账号登录。
2. 查看专家评审任务。
3. 打开项目详情和附件。
4. 填写评审意见。
5. 如页面提供评分项，按要求评分。
6. 提交评审结果。
7. 在“已审”中查看历史评审。

## 7. 管理员操作

管理员用于日常业务管理：

- 项目终审
- 验收管理
- 延期处理
- 单位注册审核
- 账号管理
- 申报批次管理
- 邮件中心
- 安全事件查看

### 7.1 申报批次管理

1. 进入“申报批次”。
2. 新增批次。
3. 设置批次名称、编号、开始时间、结束时间、开放状态。
4. 配置允许项目类别和类型。
5. 配置验收必传材料。
6. 打开批次。

### 7.2 项目终审

1. 进入“项目管理”或 Dashboard 待终审指标。
2. 筛选待终审项目。
3. 查看区县、部门、专家意见。
4. 选择通过、退回或关闭。

### 7.3 验收管理

1. 进入“验收管理”。
2. 默认查看当前待处理验收。
3. 切换“已处理”查看历史。
4. 打开验收详情。
5. 查看材料、意见、阶段条。
6. 填写终审意见并关闭验收。

## 8. 超级管理员操作

超级管理员用于系统配置和高风险运维：

- 首页管理
- 系统文案
- 角色权限
- 安全中心
- 邮件配置
- 测试数据治理

### 8.1 首页管理

路径：

```text
首页管理 -> 品牌素材
```

可维护：

- Logo
- Banner
- Favicon / 站点图标

Favicon 要求：

- 支持 `ico/png/svg`
- 建议 512KB 以内
- 上传后刷新或重新打开浏览器标签页查看效果

### 8.2 系统文案

1. 进入“系统文案”。
2. 按模块搜索文案。
3. 修改显示文字。
4. 可隐藏或恢复。
5. 如误删或禁用，前端会回退代码默认文案。

### 8.3 安全中心

可查看：

- 登录失败事件
- 登录限流事件
- 账号锁定
- IP 风控
- 白名单和临时放宽状态

常用操作：

- 按账号或 IP 查询异常登录。
- 解锁被锁定账号。
- 临时放宽登录限流用于验收。
- 配置测试白名单 IP。

### 8.4 测试数据治理

项目、验收、申报批次页面有“测试数据”筛选：

- 只看测试数据
- 排除测试数据

超管在申报批次页面可点击“归档测试批次”。

说明：

- 归档只改变测试批次状态。
- 不删除测试项目、验收和全周期记录。
- 用于避免 E2E 测试批次影响正式首页和统计。

## 9. 推荐业务验收路线

1. 超管登录，检查首页管理、安全中心、系统文案。
2. 管理员登录，检查批次、项目、验收、单位管理。
3. 单位登录，创建草稿、查看项目、进入全周期。
4. 区县登录，查看待审和已审。
5. 部门登录，查看待审和已审。
6. 专家登录，查看评审任务。
7. 回到管理员，检查终审和验收历史。
